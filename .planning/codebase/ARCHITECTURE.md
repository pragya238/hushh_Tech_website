## Application Modules

The codebase is organized into distinct product modules:

### 1. Main Website (`src/pages/`, `src/components/`)
- Marketing pages
- Investor profiles
- Onboarding flow (15 steps)
- Community features
- Service pages

### 2. Hushh AI (`src/hushh-ai/`)
- AI chat interface
- Clean architecture (core/data/domain/presentation)
- Supabase-backed conversations

### 3. Hushh Agent (`src/hushh-agent/`)
- Career/Resume node
- Phone/Email authentication
- ReAct agent pattern
- Vision-based sessions
- Emotion detection (MediaPipe)

### 4. Kai (`src/kai/`)
- Voice-based AI assistant
- Gemini Live integration
- Decision card interface

### 5. Kai India (`src/kai-india/`)
- India-specific investment advisor
- Market intelligence
- Investment calculator

### 6. Hushh Studio (`src/hushh-studio/`)
- Video generation (Veo)
- Creative tools

### 7. Hushh Intelligence (`src/hushh-intelligence/`)
- Intelligence services
- Profile search
- Data enrichment

## Data Flow

### Authentication Flow
```
User → Supabase Auth → JWT Token → Protected Routes
     ↓
Firebase (Phone Auth) → Supabase Session
     ↓
Apple/Google Sign-In → Supabase User
```

### Investor Onboarding Flow
```
Step 1-15 → Supabase onboarding_data table
         → AI Inference (DOB, Address, Profile)
         → Stripe Payment
         → Identity Verification
         → NDA Signing
         → CEO Calendar Booking
```

### AI Chat Flow
```
User Message → Edge Function → LLM (Gemini/OpenAI)
           → Response Streaming
           → Supabase Storage (History)
```

## State Management

- **Global State**: Zustand stores
- **Server State**: React Query patterns (via Supabase client)
- **Form State**: React Hook Form
- **URL State**: React Router DOM

## Deployment Architecture

```
Vercel (Frontend + API Routes)
    │
    ├── Build: Vite
    ├── CDN: Vercel Edge
    └── Functions: api/*.js

Supabase (Backend)
    │
    ├── Database: PostgreSQL
    ├── Auth: JWT + RLS
    ├── Storage: S3-compatible
    └── Edge Functions: Deno

Google Cloud (Extended Services)
    │
    └── Cloud Run: Email templates, OSINT

Mobile (Capacitor)
    │
    ├── iOS: App Store
    └── Android: Play Store
```
